### Android 配置项

复制 `example/android/scstatistics` 目录至 `your project/android/` 目录下，并在 `your project/android/settings.gradle` 中添加 `include ':scstatistics'`

* 不要修改目录名字

```groovy

include ':app'
/// 需要添加的
include ':scstatistics'

```
